# Теория графов - Учебные лабораторные работы

Этот репозиторий содержит реализацию различных алгоритмов на графах на Python, выполненных в рамках учебных лабораторных работ.

## Структура репозитория

- **1** - Базовые операции с графами и алгоритмы поиска достижимости
- **2** - Алгоритмы обхода графов (BFS, DFS) и поиска путей
- **3** - Алгоритм Дейкстры для поиска кратчайших путей
- **4** - Алгоритм Флойда-Уоршелла для поиска кратчайших путей между всеми парами вершин
- **5** - Алгоритм Форда-Беллмана для поиска кратчайших путей

## Описание лабораторных работ

### Лабораторная 1: Базовые операции с графами
- Создание матрицы смежности и списка ребер
- Визуализация графов с помощью NetworkX и Matplotlib
- Реализация алгоритмов поиска матрицы достижимости:
  - Логический алгоритм
  - Алгоритм Уоршелла
- Поиск компонент связности графа

### Лабораторная 2: Обход графов и поиск путей
- Генерация графа с заданными компонентами связности
- Реализация алгоритмов обхода:
  - Поиск в ширину (BFS)
  - Поиск в глубину (DFS)
- Поиск кратчайшего пути между вершинами
- Поиск лексикографически первого пути
- Определение компонент связности графа

### Лабораторная 3: Алгоритм Дейкстры
- Реализация алгоритма Дейкстры для поиска кратчайших путей
- Работа со взвешенными графами
- Восстановление кратчайшего пути между заданными вершинами
- Оценка времени выполнения алгоритма

### Лабораторная 4: Алгоритм Флойда-Уоршелла
- Реализация алгоритма Флойда-Уоршелла
- Поиск кратчайших путей между всеми парами вершин
- Работа с матрицами расстояний
- Сравнение времени выполнения

### Лабораторная 5: Алгоритм Форда-Беллмана
- Реализация алгоритма Форда-Беллмана
- Обнаружение циклов отрицательного веса
- Поиск кратчайших путей из одной вершины
- Анализ производительности алгоритма
